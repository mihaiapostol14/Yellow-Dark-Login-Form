# Yellow Dark Login Form

## Description

The **Yellow Dark Login Form** is a sleek, modern login interface designed with usability and aesthetics in mind. This project provides a clean, elegant form with dark-themed styling, emphasizing user experience and responsive design across devices.

## Preview
![Yellow Dark Login Form Preview](https://github.com/mihaiapostol14/Yellow-Dark-Login-Form/blob/bee4fdb7d013d75e2025fcae113121e3757bfa6c/assets/preview.png)

## Technologies Used

This project leverages the following technologies:

- **HTML (50.0%)**: The backbone of the project for structuring the web content.
- **CSS (42.7%)**: Provides design and style to the login form, ensuring responsiveness, and a visually appealing layout.
- **JavaScript (7.3%)**: Adds interactivity and functionality to the form.

## Installation

To set up this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/mihaiapostol14/Yellow-Dark-Login-Form.git
   cd Yellow-Dark-Login-Form
   ```


2. Open the `index.html` file in your browser or use a live server.

## Usage

1. Explore the login form and test its responsiveness and interactivity.
2. Customize the styling or add more features if desired.

Enjoy a beautifully crafted dark login form that stands out!
